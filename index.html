EasyStream

Native Android app with Firebase Backend

EasyStream is an app for finding and rating public restrooms. I wrote the initial version with Melanie Belkin using a MySQL and PHP backend. I have since switched to using firebase to dynamically load map markers from a dataset of over 25,000 entries using GeoFire queries, making the app useful for finding a bathroom in any major city. I've also added search and geolocation functionality.
